ETS2's DLC

Comment installer des DLC sur ETS2

1. Téléchargez ici les DLC puis cliquez sur la dernière version

2. Puis cliquez ici

3. Puis cliquez là et patientez puis cliquez sur "Download Link" puis "Download" ❗ Attention aux pub et vérifiez le nom du fichier : ETS2-MAP-DLCs-version.rar❗

4. Allez dans l'archive dans "steamapps > common > Euro truck simulator 2" mot de passe : "mosbymods.de"

5. Faites Win + R et tapez cette commande "%ProgramFiles(x86)%\steam\steamapps\common\Euro Truck Simulator 2"

6. Extraire les fichiers de l'archive dans le dossier ci-dessus et cliquez sur "remplacer les fichiers"

Lancez ETS2 et les DLC seront activés

Comment désinstaller les DLC

Les DLC doivent être désinstallés si vous voulez jouer à TruckersMP

1. Allez dans les propriétés du jeu

2. Allez dans fichiers installés

3. Cliquez sur "Vérifiez l'intégrité des fichiers du jeu"

Patientez et cebon
